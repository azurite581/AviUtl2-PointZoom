# AviUtl2 ポイントズーム

アンカーで指定したポイントにズームする [AviUtl2](https://spring-fragrance.mints.ne.jp/aviutl/) 用スクリプトです。

## 動作環境

- [AviUtl ExEdit2](https://spring-fragrance.mints.ne.jp/aviutl/)
<br>beta18 以降必須（beta18b で動作確認済み）。

## 導入方法
1. [Release](https://github.com/azurite581/AviUtl2-PointZoom/releases) から zip ファイルをダウンロードしてください。
2. zip ファイルを展開し、`ポイントズーム.anm2` を `C:\ProgramData\aviutl2\Script` フォルダか一層下にあるフォルダに入れる、または本体に D&D してください。

## 使い方
デフォルトでは `変形` カテゴリの中にあります。
<br>ズームしたい箇所にアンカーを移動させ、拡大率のパラメーターを変更してください。

## パラメーター

### トラックバー
- #### 拡大率
  拡大率。上限は 10000 です。
- #### X
  ズームする X 座標。

- #### Y
  ズームする Y 座標。

### リストボックス
- #### 領域外
  縮小したときに生じる画像の領域外のタイプを指定します。

  - クリップ：透明
  - 引き伸ばし：境界の​色
  - ループ：元画像と同じ向きでループ
  - ミラー：元画像を​反転しながらループ

### チェックボックス
- #### 補間なし
  チェックを入れると補間を​しなくなります。領域外はリストボックスの項目にかかわらず常に透明になります。

### PI
パラメーターインジェクション用の入力欄です。以下の形式に沿って値を入力することで、各種パラメーターの値を上書きできます（実際に入力するときは波括弧は不要です）。
範囲を超える値が入力された場合は自動で丸められます。

```
{ zoom, x, y, outside_region, interp }
```

|  | 説明 | 型 | 範囲 |
|:---|:---|:---|:---|
| zoom | 拡大率 |number | [0, 10000] |
| x | アンカーの x 座標 |number | [-4000, 4000] |
| y | アンカーの y 座標 | number | [-4000, 4000] |
| outside_region | 領域外のタイプ | number | [1, 4] |
| interp | 補間するかどうか | number または boolean | [0, 1] または false/true|
